To install, extract this zip into a folder and link this folder into your eclipse installation, or you may copy these files into your MCP install.
You may include these API files with your mod code, but do beware that stuff may break, and API conflicts may happen.
Feel free to inform us if you encounter any issues whatsoever (via IRC or github issues)
For reference this is ForgeEssentials API revision 2.

You can read the FE license at https://github.com/ForgeEssentials/ForgeEssentialsMain/blob/master/LICENSE.TXT .
Likewise, javadocs can be downloaded from https://github.com/ForgeEssentials/ForgeEssentialsMain/blob/master/APIjavadoc-rev1.zip
